# Projet Rentop

Rentop est le frontend d'un site de location de voiture. Projet réalisé dans le cadre d'un cours d'ui/ux.

# Guide d'installation

## Prérequis

Assurez-vous d’avoir les outils suivants installés :

| Outil            | Version recommandée |
|------------------|---------------------|
| Node.js          | 22.20.0             |
| npm              | 11.6.2              |

---

## Étapes d’installation

### 1. Cloner le dépôt

```bash
git clone https://github.com/Christmann-Julian/rentop.git
cd rentop
```
---

### 2. Installer les dépendances

```bash
npm install
```
